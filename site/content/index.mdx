# chibireact

**React を小さく作りながら学ぶ本** へようこそ。

本書は、Vue 学習書 [chibivue](https://book.chibivue.land/) の構成と思想を React 向けに再設計した、日本語の連載書籍です。「React は使えるけれど、内部の仕組みは知らない」中級者を対象に、**React 18+ の主要機能を最小実装で写経しながら理解する**ことを目的としています。

## こんな人に向いています

- React で実務はしているが、`useState` や `useEffect` の中身がブラックボックスで気持ち悪い
- Fiber や Concurrent Rendering について「聞いたことはある」止まり
- React の公式ドキュメントを読み終えて、もう一段深く潜りたい
- 写経しながら手を動かして覚えるのが好き

## 本書の特徴

| 軸 | 本書のスタンス |
|---|---|

「Build your own React」系の教材は英語圏に複数ありますが、**日本語 × 写経 × 連載書籍 × React 18+** の 4 条件を満たすものは現状ありません（[Plans/03-prior-art.md](https://github.com/tokechan/chibireact/blob/main/Plans/03-prior-art.md) で詳細）。本書はその空白地帯を埋めることを狙います。
