# chibireact へようこそ

**React を小さく作りながら学ぶ本** の準備中ページです。

## 本書について

chibireact は、Vue 学習書 [chibivue](https://book.chibivue.land/) の構成と思想を React 向けに再設計したものです。
**写経しながら React の内部実装を理解できる、日本語の連載書籍**を目指しています。

## 想定読者

- React は使えるが、内部の仕組みを知らない中級者
- Fiber / Hooks / Concurrent Rendering の動作原理を実装ベースで理解したい人
